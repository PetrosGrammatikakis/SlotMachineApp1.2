# 🎰 SlotMachineApp1.2

SlotMachineApp1.2 is a simple mobile slot machine game built using **Kotlin** in **Android Studio**, with the assistance of AI tools during development.

The app focuses on gameplay, coin progression, and basic customization through an in-game shop system.

---

## 🚀 Features

### 🎮 Main Gameplay
- Classic slot machine mechanics
- Simple and intuitive gameplay
- Coin-based reward system

### 🏠 Start Screen
- Start Game button
- Access to in-game shop

### 💰 Coin System
- Player coins are saved locally
- If the player has less than **49 coins**, they receive **50 coins the next day**
- Persistent data saving between sessions

### 🛒 Shop System
- Purchase different **background themes**
- Customize the game appearance

---

## 💾 Data Persistence
- All player data (coins, progress, and settings) is saved locally on the device
- Ensures progress is not lost after closing the app

---

## 📱 Tech Stack
- Kotlin
- Android Studio
- Local Storage (Data Persistence)
- AI-assisted development
